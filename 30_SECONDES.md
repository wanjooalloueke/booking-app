# ⚡ 30 SECONDES - PAS DE TEXTE!

**Vous êtes ici? OK, allons-y!**

---

## ✅ ÉTAPE 1 (Copier-coller dans le terminal)
```bash
npm install
```

**Attendez 30 secondes.**

---

## ✅ ÉTAPE 2 
Ouvrez MySQL et tapez:
```sql
CREATE DATABASE booking_app;
USE booking_app;
SOURCE database/schema.sql;
```

**30 secondes.**

---

## ✅ ÉTAPE 3
Trouvez le fichier `.env` et remplacez:

```
DB_PASSWORD=votre_mot_de_passe_mysql_ici
```

---

## ✅ ÉTAPE 4
Dans le terminal:
```bash
npm run dev
```

**30 secondes.**

---

## ✅ ÉTAPE 5
Ouvrez votre navigateur et collez:
```
http://localhost:5000/frontend/index.html
```
