---

## 🚀 Pour démarrer (3 étapes)

```bash
# 1. Installer
npm install

# 2. Créer la BD
mysql -u root -p
CREATE DATABASE booking_app;
USE booking_app;
SOURCE database/schema.sql;
EXIT;

# 3. Lancer
npm run dev
```

Puis ouvrir : `http://localhost:5000/frontend/index.html`
